# Library System

This is a program that allows you to manage a library system. It allows you to create library accounts, view those records, and manage books (returns, availability, and borrowing). 

## Instructions for Build and Use

Steps to build and/or run the software:

1. To run the software, it is necessary to have Python installed as it is a python program. 
2. Clone the repository and open the librarySystem.py file.  

Instructions for using the software:

1. Make sure to pay attention to the accountID given when you create an account. It is necessary to know this infromation to be able to use the other functions in the program. 
2. This program is case sensitive so you must write book titles exactly as you see it presented. If not, then the program will not recognize that you are trying to refer to the book in the database. 
3. Read the instructions given for each step, they tell you what information you must provide in order to use the program.
